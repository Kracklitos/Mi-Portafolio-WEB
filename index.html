Code Quest - Cuestionario de Programación

Este proyecto es un cuestionario interactivo de programación desarrollado completamente en Java utilizando la biblioteca Swing. El cuestionario ofrece una serie de preguntas desafiantes (no tanto) que cubren diferentes áreas de la programación, con el objetivo de evaluar los conocimientos y habilidades de los usuarios.

Este proyecto me proporcionó una valiosa experiencia en el desarrollo de aplicaciones con Java Swing, permitiéndome dominar el diseño de interfaces de usuario interactivas, la creación de eventos, el manejo de componentes y la gestión de la lógica del juego. Aprendí a utilizar una biblioteca externa para integrar sonido, lo que enriqueció la experiencia del usuario. Además, este proyecto me desafió a resolver problemas complejos y a trabajar de manera organizada y eficiente, mejorando mis habilidades de desarrollo de software.

App de Análisis del Desempeño Productivo de una Empresa

Este proyecto es una aplicación de análisis de desempeño productivo para una empresa, desarrollada en C usando el QT.La aplicación permite a los usuarios ingresar datos de producción, realizar cálculos de rendimiento y generar informes que ayudan a la toma de decisiones.

Este proyecto no solo me brindó la oportunidad de practicar mis habilidades de programación, sino que también me enseñó la importancia de la planificación y el diseño antes de comenzar a codificar. Aprendí a gestionar errores y validar datos para asegurar la calidad de la información procesada. Además, adquirí experiencia en el uso de estructuras de datos básicas y en la implementación de algoritmos simples. .
